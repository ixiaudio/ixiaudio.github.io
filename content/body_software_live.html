Ah! live performances. so much can go wrong when playing live - the last thing you need is the software crashing every 2 minutes. But there's no way around this - anyone who says otherwise is lying!

We have tried to build the ixi software to be loaded and played without straining your machine (check the minimal specs on the download page). This means that it will work smoothly in a live performance situations.


So why would i want to use an ixi app live?
As laptop computers have become cheaper we've seen more and more laptop performances. Though no different to a DJ playing out, the audience always seem to be faced with a musician sitting behind a black book with a glowing apple mac icon - wondering what the hell he/she is doing.

With the current ixi apps - the performance is lived out through the interface components. If you rigg up a projector to the laptop, the audience can see you move the interface components around the screen and understand how you are in control of the medium. We're not saying that this is fascinating viewing for a 5 hour set, but definitely more inviting that what we've seen before!


How do I record my performances?
We recommend that in a live performance situation that you run as few applications as possible to reduce the chance of a system crash. So to record your performance we recommend that you connnect a recording device (eg. DAT or mini disc) to the outbound mixing desk or PA system.
