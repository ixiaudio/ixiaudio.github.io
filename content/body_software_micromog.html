The MicroMog is a small machine for sound terrorists. It is thought of as a performance tool which can be used together with other software applications. Works as some kind of arpeggiator

In the MicroMog, the buttons triggered create layers of sounds. The relation between the sounds can be adjusted (delay relation, pitch relation and number of notes responding to each stroke). It is not musical program (in the traditional sense), even though it is visually camouflaged as a "typical" small sound software. It has been designed for live performance and therefore it is totally controllable with the (standard) keyboard and the mouse.
